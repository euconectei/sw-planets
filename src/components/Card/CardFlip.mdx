# CardFlip

CardFlip allows you to display 2 different components with a rotating effect.

## Properties

<Props of={CardFlip} />

## Basic Usage

<Playground>
  {() => {
    const [flip, setFlip] = React.useState(false);
    return (
      <div style={{ backgroundColor: '#ccc', height: 500, width: 300}}>
        <button onClick={() => setFlip(!flip)}>Rotate</button>
        <CardFlip flipped={flip} back={<Loading />} front="Front" />
      </div>
    );
  }}
</Playground>
